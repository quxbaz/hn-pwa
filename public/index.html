This is a test.

Each comments link below is pre-cached in the background by a service worker.

When you click a comments link, it should load instantly.

Story links will load normally (ie, not instantly). That's because service workers are scoped by domain. Any resource you cache will only be shared by browser tabs sharing the same domain. When you navigate to a different URL, the service worker will no longer be operating within that context.

It's possible that there might be a workaround to this.
